# AI Board Frontend

This folder contains the frontend code for the AI Board of Directors application.

## Structure
- `src/` - React/Vue components
- `public/` - Static assets
- `components/` - UI components

## Setup
```bash
npm install
npm run dev
```

## Tech Stack
- React or Vue.js (framework to be determined)
- TailwindCSS for styling
- TypeScript for type safety
